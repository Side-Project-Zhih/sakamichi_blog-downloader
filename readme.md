1. 下載 node.js 選擇 stable 版本
2. 打開 cmd
3. 輸入 "cd " 再把整個資料夾拉進 cmd 按 enter
4. 輸入 "npm install" 安裝 modules
5. 裝完後輸入 "node fileName.js" 就會執行

== fileName===>為裡面 js 檔的檔名
== 櫻坂賀卡可直接執行，輸入 " node 檔名.js " 執行
